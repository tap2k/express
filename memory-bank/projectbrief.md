# Project Brief: Express CMS

**Project Title:** Express CMS - Multimedia Sharing

**1. Introduction**

Express CMS shares multimedia content using Maps, Slideshows, and Grids. Users upload media, annotate, assign map locations, and create videos. Built with NextJS, Reactstrap, and Strapi.

**2. Core Features**

*   **Content Formats:** Maps, Slideshows, Grids.
*   **Media Upload:** User media upload.
*   **Annotation:** Media annotation.
*   **Map Locations:** Assign locations to media for maps.
*   **Video Generation:** Create composited videos.

**3. Technical Requirements**

*   **Frontend:** NextJS, Reactstrap.
*   **Backend:** Strapi.
